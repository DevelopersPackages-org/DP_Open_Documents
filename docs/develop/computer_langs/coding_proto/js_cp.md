## 一般

Javascript はブラウザサイドにのみ用い、Node.js 等は用いてはならない。サーバサイドやアプリケーション等は適宜 Dart 等を使用する。

## 変数宣言

変数宣言時は必ず let を用いるものとし、いついかなる時も var を用いてはならない。
